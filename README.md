# Static Analysis Lab

## Deskripsi
Repository ini berisi dokumentasi hasil analisis statis terhadap file Portable Executable (PE) sebagai bagian dari pembelajaran Reverse Engineering.

## Tujuan
- Mempelajari struktur file PE.
- Melakukan analisis statis tanpa menjalankan program.
- Mendokumentasikan hasil analisis menggunakan tools Reverse Engineering.

## Tools yang Digunakan
- PEStudio
- Ghidra
- Detect It Easy (DIE)

## Isi Repository
Repository ini berisi laporan singkat hasil analisis statis terhadap beberapa file binary PE.

## Disclaimer
Seluruh analisis dilakukan untuk tujuan edukasi dan penelitian pada lingkungan Virtual Machine (VM). Repository ini tidak mendistribusikan malware aktif.
